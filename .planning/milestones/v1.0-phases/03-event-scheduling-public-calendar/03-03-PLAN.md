Create `src/utils/calendarHelpers.ts` with helper functions for calendar data formatting.

Import from date-fns: `startOfMonth`, `endOfMonth`, `startOfWeek`, `endOfWeek`, `eachDayOfInterval`, `format`, `isSameDay`, `isSameMonth`, `getDay`.
Import from date-fns-tz: `formatInTimeZone`.

Define types:

```typescript
export interface CalendarEvent {
  id: number | null;               // null for virtual recurring instances
  title: string;                   // route name
  category: string;                // route category
  startDateTime: string;           // ISO 8601 string
  displayDate: string;             // "Mon, Feb 17, 2026"
  displayTime: string;             // "6:30 PM"
  endLocation: string | null;
  notes: string | null;
  isRecurring: boolean;            // true if from recurring template
  recurringTemplateId: number | null;
}

export interface MonthDay {
  date: string;           // "2026-02-17"
  dayOfMonth: number;     // 17
  isCurrentMonth: boolean;
  isToday: boolean;
  events: CalendarEvent[];
}

export interface MonthGrid {
  year: number;
  month: number;
  monthName: string;       // "February 2026"
  weeks: MonthDay[][];     // Array of 5-6 weeks, each week is 7 days
}
```

Functions:

`formatEventForCalendar(event, timezone?)`:
- Accept an event object (from eventService or recurringService virtual event)
- Default timezone to 'America/New_York' (Astoria is NYC)
- Return CalendarEvent with:
  - title: event.route.name (or event.routeName if denormalized)
  - category: event.route.category
  - startDateTime: event.startDateTime ISO string
  - displayDate: format with date-fns `format(date, 'EEE, MMM d, yyyy')`
  - displayTime: format with date-fns `format(date, 'h:mm a')`
  - isRecurring: !!event.recurringTemplateId
  - Other fields mapped directly

`buildMonthGrid(year, month, events)`:
- Compute start of month and end of month for the given year/month
- Extend to full weeks: startOfWeek(startOfMonth) through endOfWeek(endOfMonth)
- Generate each day using eachDayOfInterval
- Group days into weeks (arrays of 7)
- For each day, filter events that match (using isSameDay)
- Return MonthGrid object with monthName formatted as "February 2026"

`mergeAndSortEvents(dbEvents, recurringInstances)`:
- Accept array of DB events (from eventService.listEvents) and recurring instances (from recurringService.getAllInstancesInRange)
- Deduplicate: if a DB event has a recurringTemplateId matching a virtual instance on the same date, keep the DB event (it may have been individually edited)
- Combine remaining, sort by startDateTime ascending
- Return unified CalendarEvent[] array
