**1. Create `src/middleware/validate.ts`:**

Reusable Zod validation middleware for body and query params:

```typescript
import { Request, Response, NextFunction } from 'express';
import { ZodSchema, ZodError } from 'zod';

export function validateBody(schema: ZodSchema) {
  return (req: Request, res: Response, next: NextFunction) => {
    try {
      req.body = schema.parse(req.body);
      next();
    } catch (error) {
      if (error instanceof ZodError) {
        res.status(422).json({
          error: 'Validation failed',
          details: error.errors.map(e => ({
            field: e.path.join('.'),
            message: e.message,
          })),
        });
        return;
      }
      next(error);
    }
  };
}

export function validateQuery(schema: ZodSchema) {
  return (req: Request, res: Response, next: NextFunction) => {
    try {
      req.query = schema.parse(req.query) as any;
      next();
    } catch (error) {
      if (error instanceof ZodError) {
        res.status(422).json({
          error: 'Validation failed',
          details: error.errors.map(e => ({
            field: e.path.join('.'),
            message: e.message,
          })),
        });
        return;
      }
      next(error);
    }
  };
}
```

**2. Create `src/routes/routes.ts`:**

Follow the auth.ts pattern (Router, asyncHandler). Import from service and validation:

```typescript
import { Router } from 'express';
import { asyncHandler } from '../utils/asyncHandler.js';

import { createRouteSchema, updateRouteSchema, listRoutesQuerySchema } from '../validation/routes.js';

```

All routes on this router require auth. Apply `requireAuth` to the entire router:

```typescript
const router = Router();
router.use(requireAuth);
```

Implement these endpoints:

**POST /** (Create route)
- Middleware: `validateBody(createRouteSchema)`
- Call `routeService.createRoute(req.body)`
- Return 201 with `{ route: result }`

**GET /** (List routes)
- Middleware: `validateQuery(listRoutesQuerySchema)`
- Extract `category` from `req.query`
- Call `routeService.listRoutes(category ? { category } : undefined)`
- Return 200 with `{ routes: result }`

**GET /:id** (Get single route)
- Parse `req.params.id` with `parseInt`, return 400 if NaN
- Call `routeService.getRouteById(id)`
- If null, return 404 `{ error: 'Route not found' }`
- Return 200 with `{ route: result }`

**PUT /:id** (Update route)
- Parse `req.params.id` with `parseInt`, return 400 if NaN
- Middleware: `validateBody(updateRouteSchema)`
- Check that at least one data field (name, distance, category, endLocation) is present beyond version. If only version provided with no data fields, return 422 `{ error: 'At least one field to update is required' }`
- Call `routeService.updateRoute(id, req.body)`
- If result has `error: 'not_found'`, return 404 `{ error: 'Route not found' }`
- If result has `error: 'conflict'`, return 409 `{ error: 'Route was modified by another user. Please refresh and try again.' }`
- Return 200 with `{ route: result.route }`

**DELETE /:id** (Delete route)
- Parse `req.params.id` with `parseInt`, return 400 if NaN
- Call `routeService.deleteRoute(id)`
- If result has `error: 'not_found'`, return 404 `{ error: 'Route not found' }`
- If result has `error: 'conflict'`, return 409 `{ error: 'Route is referenced by events and cannot be deleted' }`
- Return 204 No Content (no body)

Export default router.
