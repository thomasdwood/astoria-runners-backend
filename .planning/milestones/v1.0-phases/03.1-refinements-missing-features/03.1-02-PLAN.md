       - `getLocationSuggestions()`: Run a UNION of DISTINCT non-null/non-empty startLocation and endLocation values from both routes and events tables. Return sorted deduplicated array of strings. Use raw SQL via `db.execute(sql\`...\`)` or multiple queries merged in JS.

       - POST `/` — requireAuth, validateBody(createCategorySchema), create, return 201 with `{ category }`
       - PUT `/:id` — requireAuth, validateBody(updateCategorySchema), update, return 200 or 404
       - DELETE `/:id` — requireAuth, delete. If error='in_use', return 409 with `{ error: 'Cannot delete category', routeCount }`. If success, return 200. If not found, return 404.

    5. Create `src/routes/settings.ts`:

       - GET `/locations` — NO auth required. Return `{ locations: string[] }` from getLocationSuggestions()
       - PUT `/:key` — requireAuth, validate body has `{ value: string }`, upsert, return 200
