- `validateQuery` middleware's `Object.assign(req.query, parsed)` doesn't reliably coerce types — Express req.query may keep string types. Fixed at usage site with explicit `Number()` coercion in calendar route
- Schema `.js` extensions removed from `src/db/schema/` imports for drizzle-kit CJS compatibility
