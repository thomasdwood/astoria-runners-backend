       - `useUpdateCategory()`: useMutation, PUT /api/categories/:id, invalidate ['categories'] on success

    2. Create `client/src/hooks/use-settings.ts`:
       - `useSettings()`: useQuery with queryKey ['settings'], GET /api/settings, select data.settings

       - `useUpdateSetting()`: useMutation, PUT /api/settings/:key with body { value }, invalidate ['settings'] on success
       - `useLocationSuggestions()`: useQuery with queryKey ['location-suggestions'], GET /api/settings/locations, select data.locations

    3. Update `client/src/hooks/use-routes.ts`:
