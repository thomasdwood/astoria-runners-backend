  <action>
    1. Create src/config/session.ts:
       - Import express-session, RedisStore from connect-redis, Redis from ioredis
       - Create Redis client using config.redis.url (from env.ts)
       - Handle Redis connection errors gracefully (log error, don't crash)
       - Configure session middleware:
         - store: RedisStore with Redis client, prefix 'astoria:sess:'
         - secret: config.session.secret
         - name: 'sid' (NOT default 'connect.sid' -- security per research)
         - resave: false
         - saveUninitialized: false
         - cookie:
           - secure: config.nodeEnv === 'production' (HTTPS only in prod)
           - httpOnly: true (prevent XSS access)
           - sameSite: 'lax' (CSRF protection)
           - maxAge: 7 * 24 * 60 * 60 * 1000 (7 days -- per user decision)
       - Export sessionMiddleware

    2. Create src/services/authService.ts:
       - Import { hash, verify } from '@node-rs/argon2'
       - Import { db } from config/database.ts, { users } from db/schema
       - Import { eq } from 'drizzle-orm'

       - hashPassword(password: string): Hash with argon2, memoryCost: 19456, timeCost: 2, parallelism: 1
       - verifyPassword(hash: string, password: string): Verify with argon2
       - validatePassword(password: string): Returns { valid: boolean, message?: string }
         - Minimum 8 characters (NIST recommendation per research)
         - No complexity requirements (NIST recommends against them)
         - Return descriptive message if invalid: "Password must be at least 8 characters"
       - findUserByEmail(email: string): Query users table by email, return user or null
       - Export all functions

    3. Create src/routes/auth.ts:
       - Import Router from express, asyncHandler, authService functions
       - Create router

       POST /login:
       - Extract email, password from req.body
       - Input validation: both fields required (return 400 if missing)
       - Call findUserByEmail(email)
       - If no user: return 401 { error: 'Invalid credentials' } (LOCKED DECISION -- generic error)
       - Call verifyPassword(user.passwordHash, password)
       - If invalid: return 401 { error: 'Invalid credentials' } (same generic error -- don't reveal if email exists)
       - Set req.session.userId = user.id
       - Set req.session.email = user.email
       - Set req.session.displayName = user.displayName
       - Return 200 { success: true, user: { id, email, displayName } }

       POST /logout:
       - Call req.session.destroy() (LOCKED DECISION -- global logout, kills session)
       - Clear cookie: res.clearCookie('sid')
       - Return 200 { success: true }
       - Handle destroy errors: return 500

       GET /me:
       - If req.session.userId: return user info from session
       - If not: return 401

       - Export router
  </action>
